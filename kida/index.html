TAG 1

Python Data Science Bootcamp

Von Pandas bis Machine Learning

1 Tag (8 Stunden)
Einsteiger & Fortgeschrittene

Kursinhalte

Python Grundlagen
  • • Python Syntax & Best Practices
  • • Datenstrukturen & Funktionen
  • • NumPy für numerische Berechnungen
Datenanalyse mit Pandas
  • • DataFrames & Datenmanipulation
  • • Datenbereinigung & -transformation
  • • Explorative Datenanalyse (EDA)
Visualisierung
  • • Matplotlib & Seaborn
  • • Interaktive Plots mit Plotly
  • • Storytelling mit Daten
Machine Learning Basics
  • • Scikit-learn Einführung
  • • Supervised Learning (Regression, Klassifikation)
  • • Model Training & Evaluation
Lernziele

Nach diesem Kurs können Sie eigenständig Datenanalysen durchführen, aussagekräftige Visualisierungen erstellen und einfache Machine Learning Modelle trainieren. Ideal für Ingenieure, Analysten und alle, die datengetriebene Entscheidungen treffen wollen.

TAG 2

Python Engineering Praxis

Testing, CI/CD & Deployment

1 Tag (8 Stunden)
Entwickler mit Python-Grundlagen

Kursinhalte

Clean Code & Best Practices
  • • PEP 8 & Code-Konventionen
  • • Design Patterns in Python
  • • Code-Refactoring Strategien
Testing & Quality Assurance
  • • Unit Testing mit pytest
  • • Test-Driven Development (TDD)
  • • Mocking & Test Coverage
Version Control & Collaboration
  • • Git Workflows (Feature Branches, PRs)
  • • Code Reviews Best Practices
  • • Dokumentation mit Sphinx
CI/CD & Deployment
  • • GitHub Actions / GitLab CI
  • • Docker für Python Apps
  • • Package Management & Distribution
Lernziele

Verwandeln Sie Ihre Python-Skripte in professionelle, wartbare Software. Sie lernen industrielle Best Practices für Testing, Deployment und Zusammenarbeit im Team kennen. Perfekt für den Übergang von Prototypen zur Produktionsreife.

Prompt Engineering

AI-Coding Tools
  • • GitHub Copilot Best Practices

Agentic AI Workflows

Maximieren Sie Ihre Produktivität mit modernen KI-Assistenten. Lernen Sie, wie Sie ChatGPT, Copilot und andere Tools effektiv in Ihren täglichen Entwicklungsprozess integrieren. Von Prompt Engineering bis hin zu automatisierten Workflows – werden Sie zum AI-Power-User.

2 TAGE

Data Science Deep Dive

Von der Datenanalyse bis zum Deployment

2 Tage (16 Stunden)
Fortgeschrittene mit ML-Grundlagen

Kursinhalte

Advanced ML Techniken
  • • Feature Engineering & Selection
  • • Ensemble Methods (Random Forest, XGBoost)
  • • Hyperparameter Tuning & AutoML
Deep Learning Basics
  • • Neural Networks mit PyTorch/TensorFlow
  • • CNNs für Computer Vision
  • • Transfer Learning & Fine-Tuning
MLOps & Productionization
  • • Model Versioning & Experiment Tracking
  • • Model Deployment (FastAPI, Docker)
  • • Monitoring & A/B Testing
Real-World Case Studies
  • • End-to-End Projekt: Von Daten zu Deployment
  • • Autonomes Fahren: Computer Vision Pipeline
  • • Prädiktive Wartung in der Industrie
Lernziele

Meistern Sie den kompletten Data Science Lifecycle – von der Problemdefinition über Feature Engineering und Model Training bis hin zum produktiven Deployment. Anhand echter Projekte aus autonomem Fahren und industrieller Anwendung lernen Sie, wie professionelle ML-Systeme gebaut werden.

Warum KIDA Akademie?

Von Forschern, die wissen, wie es in der Praxis funktioniert

Forschungsexzellenz

Dozenten mit PhD-Abschluss von FZI und KIT. Profitieren Sie von aktuellster Forschung und jahrelanger Projekterfahrung in autonomen Systemen und KI.

Praxisorientiert

Hands-on Workshops mit echten Projekten. Keine reine Theorie – Sie arbeiten an konkreten Aufgabenstellungen und können das Gelernte sofort anwenden.

Kleine Gruppen

Maximal 12 Teilnehmer pro Kurs. So können wir individuell auf Fragen eingehen und intensive Betreuung während der praktischen Übungen garantieren.

Aktuelle Technologien

Neueste Tools und Frameworks. Von PyTorch über Docker bis zu GitHub Copilot – Sie lernen mit dem State-of-the-Art Tech-Stack.

Eigene Projekte mitbringen

Bringen Sie Ihre echten Herausforderungen mit! Wir helfen Ihnen, konkrete Probleme aus Ihrem Arbeitsalltag zu lösen. Inklusive Teilnahmezertifikat und relevanten Code-Beispielen.

Firmen-Workshops

Maßgeschneiderte Inhouse-Schulungen. Wir passen Inhalte und Zeitplan an Ihre spezifischen Anforderungen und Ihr Unternehmenskontext an.

Unsere Vision

Wir haben beide unsere Promotion am FZI und KIT absolviert und jahrelang an der Spitze der KI-Forschung gearbeitet. Jetzt möchten wir unser Wissen weitergeben und Fachkräften helfen, die Skills für die KI-getriebene Zukunft zu entwickeln. Bei KIDA Akademie verbinden wir wissenschaftliche Exzellenz mit praktischer Anwendbarkeit – damit Sie nicht nur verstehen, wie moderne KI funktioniert, sondern auch wie Sie sie erfolgreich einsetzen.
